## 💡 핵심 기능
{: #core-features }

### 🗣️ AI 보이스팩 생성
{: #feature-voicepack }

- 사용자가 음성을 업로드하면 AI가 해당 음색·억양을 학습하여 보이스팩을 생성합니다.  

### 🛍️ 보이스 마켓플레이스
{: #feature-market }

- 생성한 보이스팩을 크레딧 기반으로 자유롭게 판매할 수 있고, 또한 다른 사람들의 보이스팩을 구매할 수 있습니다.  

### 🎮 보이스팩 플레이그라운드
{: #feature-playground }

- 구매한 보이스팩은 플레이그라운드에서 다양한 방식으로 활용할 수 있습니다.
